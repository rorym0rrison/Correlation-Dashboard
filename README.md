# Correlation-Dashboard
A desktop application for analysing the statistical relationship between two financial assets.

## Features
- Fetches historical price data via Yahoo Finance (yfinance)
- Computes Pearson correlation of log returns
- Fits and displays a linear regression with R²
- Plots price timeseries and regression scatter side-by-side
